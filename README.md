# Ultimate Kitchen Manager

- `defaults/` → default configuration for new events
- `extract/` → recipe, ingredient extraction from the old kitchen Excel
- `src/`, `templates/` source for the flask app
- [Handbook](handbook.md): WIP specification

# Usage

- install https://astral.sh/
- `git clone` the repo
- cd `ultimate-kitchen-manager` 
- run `uv run flask run` to start the flask development server (run the application; doesn't do much for now)
- run `uv run er.py` to generate er diagram
